# Image Assets Guide

This directory contains all image assets for the website. You can replace these images with your own following the guidelines below.

## Directory Structure

```
images/
├── team/           # Team photos for About section
├── services/       # Service-specific images
├── gallery/        # Before/after project photos
├── hero/           # Hero section backgrounds
├── testimonials/   # Customer profile photos
├── badges/         # Trust badges and certifications
└── cta/            # Call-to-action banner backgrounds
```

## Image Specifications

### Team Photos (`team/`)

- **File:** `team-photo.jpeg`
- **Used in:** About section
- **Recommended size:** 1200-1920px wide
- **Aspect ratio:** 21:9 (ultra-wide)
- **Format:** JPEG
- **Max file size:** 500KB

### Hero Backgrounds (`hero/`)

- **File:** `hero-background.jpeg`
- **Used in:** Hero section at top of homepage
- **Recommended size:** 1920px wide
- **Aspect ratio:** 16:9 or 21:9
- **Format:** JPEG
- **Max file size:** 800KB

### Service Images (`services/`)

- **Files needed:**
  - `plumbing.jpeg` - Plumbing service
  - `hvac.jpeg` - HVAC service
  - `electrical.jpeg` - Electrical service
  - `roofing.jpeg` - Roofing service
- **Recommended size:** 800-1200px wide
- **Aspect ratio:** 4:3 or 16:9
- **Format:** JPEG
- **Max file size:** 300KB each

### Gallery Images (`gallery/`)

- **Files needed (before/after pairs):**
  - `kitchen-before.jpeg` & `kitchen-after.jpeg` - Kitchen renovation
  - `hvac-before.jpeg` & `hvac-after.jpeg` - HVAC installation
  - `electrical-before.jpeg` & `electrical-after.jpeg` - Electrical panel
  - `roof-before.jpeg` & `roof-after.jpeg` - Roof replacement
  - `bathroom-before.jpeg` & `bathroom-after.jpeg` - Bathroom remodel
  - `smarthome-before.jpeg` & `smarthome-after.jpeg` - Smart home wiring
- **Recommended size:** 800-1200px wide
- **Aspect ratio:** 4:3
- **Format:** JPEG
- **Max file size:** 300KB each

### Testimonial Photos (`testimonials/`)

- **Files needed:**
  - `sarah-johnson.jpeg` - Customer testimonial
  - `mike-patterson.jpeg` - Customer testimonial
  - `jennifer-lee.jpeg` - Customer testimonial
- **Recommended size:** 400px square
- **Aspect ratio:** 1:1 (square)
- **Format:** JPEG
- **Max file size:** 100KB each

### Trust Badges (`badges/`)

- **Files needed:**
  - `bbb-accredited.jpeg` - BBB Accreditation badge
  - `licensed-insured.jpeg` - Licensed & Insured badge
  - `award-winning.jpeg` - Award Winner badge
- **Recommended size:** 200px wide
- **Format:** JPEG or PNG (PNG if transparency needed)
- **Max file size:** 50KB each

### CTA Backgrounds (`cta/`)

- **File:** `cta-background.jpeg`
- **Used in:** Call-to-action banner
- **Recommended size:** 1920px wide
- **Aspect ratio:** 21:9
- **Format:** JPEG
- **Max file size:** 500KB

## Optimization Tips

1. **Compress images** before uploading using tools like:

   - TinyPNG (https://tinypng.com)
   - Squoosh (https://squoosh.app)
   - ImageOptim (Mac)

2. **Use appropriate formats:**

   - JPEG for photographs
   - PNG for graphics with transparency
   - WebP is automatically generated by the build process

3. **Resize images** to recommended dimensions before uploading

   - Don't upload 4000px images if you only need 1200px

4. **Aim for file sizes:**
   - Hero/backgrounds: 500-800KB max
   - Service/gallery images: 200-300KB max
   - Profile photos: 50-100KB max
   - Badges: 20-50KB max

## File Naming Convention

- Use lowercase letters
- Use hyphens instead of spaces
- Use descriptive names: `team-photo.jpeg` not `img1.jpeg`
- Keep names consistent with this guide

## Replacing Images

1. Place your new image file in the appropriate directory
2. Name it exactly as shown in this guide (or update `config/content.json`)
3. Run `npm run build:optimized` to optimize the images
4. The site will automatically use your new images

## Automatic Optimization

When you run `npm run build:optimized`, the build process automatically:

- Converts images to WebP format for modern browsers
- Generates multiple responsive sizes
- Creates blur placeholders for better loading experience
- Optimizes file sizes without quality loss

## Questions?

If you need different image specifications or have questions about which images to use where, refer to the `config/content.json` file to see how images are referenced throughout the site.
