This folder contains Screaming Frog SEO outputs.

issues_overview_report.csv:
- Exported from Screaming Frog “Issues Overview”
- Each row represents an issue type, not a URL
- Columns include:
  - Issue Name
  - Issue Type (Critical / High / Medium / Low)
  - Number of URLs Affected
  - Description

Treat this file as authoritative for SEO problems.
Assume issues are systemic unless proven otherwise.
Prefer template or component-level fixes.
